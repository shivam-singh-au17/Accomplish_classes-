Print Function

 

        Learn how print function work in Python =>

        Definition and Usage:
                              The print() function prints the specified message to the 
                              screen, or other standard output device.
                              The message can be a string, or any other object, the object will 
                              be converted into a string before written to the screen.

        Syntax & Example:
                          print(object(s), sep=separator, end=end, file=file, flush=flush)

                          print("Hello", "how are you?")

        Output:
                Hello", "how are you?
