Pop Function

 

        Learn how pop function work in Python =>

        Definition and Usage:
                              The pop() method removes the specified item from the dictionary.

        Syntax & Example:
                          dictionary.pop(keyname, defaultvalue)
                          
                          car = {
                            "brand": "Ford",
                            "model": "Mustang",
                            "year": 1964
                          }
                          x = car.pop("model")
                          print(x)

        Output:
                Mustang
