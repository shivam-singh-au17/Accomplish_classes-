Type Function

 

        Learn how type function work in Python =>

        Definition and Usage:
                              The type() function returns the type of the specified object.

        Syntax & Example:
                          type(object, bases, dict)

                          a = ('apple', 'banana', 'cherry')
                          b = "Hello World"

                          x = type(a)
                          y = type(b)

        Output:
                class 'tuple'
                class 'str'
